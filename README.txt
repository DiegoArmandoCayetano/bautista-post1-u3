Descripción

Este proyecto consiste en el desarrollo de una página web de perfil personal utilizando HTML5 y CSS3.
La página incluye una sección de perfil, una lista de habilidades y un formulario de contacto.

En el desarrollo se aplican conceptos como:

Selectores CSS

Box Model con box-sizing: border-box

Posicionamiento CSS (fixed, relative, absolute)

Estilos de formulario con :focus y :hover
